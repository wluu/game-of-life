How to Run
  1. Left-click (or left-click + hold down + drag) on the board to add new cells to the board.
  2. Or, select an initial set of seed cells from the "Select seed" dropdown list to populate the board.
    3. Note: This will clear the board of any initial seed cells.
  3. Click on the "Play" button to run the game automatically. Or, click on the "Next" button to see one generation per cycle.
More on Controls
  • To remove a cell, click on the cell again. To remove multiple cells, hold down shift key + hold down on left-click + drag.
  • "Clear" button will clear the board. You can clear the board when the game is running.
  • "Stop" button will pause the game. In this state, you can add more cells to the board.
  • You cannot add more cells when the game is running.
Rules
  1. Any live cell with fewer than two live neighbors dies, as if by under population.
  2. Any live cell with two or three live neighbors lives on to the next generation.
  3. Any live cell with more than three live neighbors dies, as if by overpopulation.
  4. Any dead cell with exactly three live neighbors becomes a live cell, as if by reproduction.

Source:Conway's Rules

Notes
  • One difference between this game and Conway's is that in his game, the board's lengths are infinite. So in this game, when a cell reaches the boarder, it will die.
  • The game "ends" when all cells die off.
